하이퍼링크 관련 태그

        - 현재 HTML 문서와 다른 HTML 문서를 연결시키거나 특정 주소로 이동할 수 있도록 만드는 태그
        - a 태그로 감싸진 글, 내용, 이미지와 같은 형식은 모두 이동이 가능함
        - a 태그 + 이미지 이동 - 이미지를 클릭했을 때 이동할 수 있음

        a 태그는 특정 주소만 이동이 가능한 것이 아니라 개발자가 만든 문서파일로 이동 가능
